📖 How to Use StoryMinerPro

  1. Open this HTML file by double-clicking it in your /VIEW folder.
  2. The app will auto-load default-corpus.json from the same folder.
  3. Use the search bar and filter toggles to navigate stories.
  4. Click any card to open the Scene Reader with side-by-side wording comparison.
  5. Upload files (JSON/TXT/MD/RTF) using the "Upload" button — they'll merge into the grid instantly.
  6. Drag-drop images onto any story card to attach them (stored as base64).
  7. Press Shift + ? to view keyboard shortcuts.
  8. Adjust theme, column count, and sidebar collapse in Settings (gear icon).

All changes persist via localStorage. Refresh to reload default corpus.
