Workflow Guide

How to use StoryMinerPro efficiently

🔄 Data Flow
Library ↔ Ingest ↔ Recombine ↔ Download stories.json
All data lives in localStorage.workbench_data_v1.
Sprint 1 – Library
Drop RTF files (or use mock data) → stories appear. Use filters to narrow, then click a story to select it.
Sprint 2 – Ingest
Select a story → fill setup, consent, seed mode, explicitness, tone & DNA chips, summary, match notes. Save. Scenes and extractions store reusable building blocks.
Sprint 3 – Recombine
Set target seed/tone/DNA, click “Suggest modules” or manually add story/scene/extraction modules. Build concept stack → export markdown prompt.
Generator Modes
Faithful Zlata Match (strictest), Romantic Variation (warmer), Training-Focused (coaching), Voyeuristic Discovery (observer), Chemistry Slow-Burn (emotional arc).
